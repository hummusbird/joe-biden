# Joe Biden.

A discord bot dalai API (for entertainment purposes.)

## setup:

### install:

- node
- npm

### setup:
put your token in a new file named .env:

```
BOT_TOKEN=XXXXXXXX
```

```
npm i

npx dalai llama/alpaca install 7B
```

### run:

```
npx dalai serve

node bot.js
```
